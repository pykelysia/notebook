## ***GoLang + 原生***

### 安装流程(install)

(以0.1.0为例)

下载合适源代码版本并解压。

```
cd notebook-0.1.0
```

安装相应依赖并build。

```
go mod tidy
go build main.go
```

后续根据需要自行操作即可。

### 版本说明

0 . * . * 表示为正式版本版本号

0 . * . * (v*) 表示测试版本号

### 注意(notice)

默认开启端口8089，如需修改，可前往 notebook/config/config.go 修改。
